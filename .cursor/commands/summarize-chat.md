# 总结本对话窗口的工作

根据**当前 Cursor 对话窗口**里已完成的内容写极简回顾。不要虚构；信息不足则如实说明。

## 你必须输出的格式（仅此结构）

```text
标题：<一句话总结本次对话完成的核心工作>

内容：
## <第一处最重要改动 — 小节标题>
<极简说明：做了什么、为何重要；2–4 句>

## <第二处改动 — 小节标题>
<若无第二项重要改动则整节省略>

## <第三处改动 — 小节标题>
<若无第三项则整节省略>
```

## 约束

- **内容**下**最多 3 个小节**，每节一项「最重要」的改动、决策或交付；按重要性排序。
- 改动含：代码/配置变更、架构或流程结论、关键命令与结果、明确推迟或否决的结论。
- 全文极简：无开场白、无收尾客套；小节内短句、高密度。
- 若仅为问答无落地改动：标题概括主题，**内容**里只保留 **1** 个小节写核心结论即可。
